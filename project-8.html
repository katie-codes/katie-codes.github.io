DATA PROCESSING

The SheCodes Python assignment was to take a JSON file of weather data and process it into something readable for humans. The learning objectives was to create and run small Python scripts utilising print statements, variables, if statements, lists and simple mathematics, understand that there are different data types available in Python, and when to use each one, and understand how Python can be used to implement simple logic. https://github.com/katie-codes/python_project_part_1
